This folder stores name materials for the conjoint experiment.

Source split:
- `black_*` and `white_*` are extracted from Appendix B of `SYSTEMIC DISCRIMINATION AMONG LARGE U.S. EMPLOYERS.pdf`.
- `latino_*` and `asian_chinese_*` are newly designed name pools to extend the experiment.

Design choices:
- Black, White, and Latino names are stored with gender-split first names plus race-specific last names.
- Asian (Chinese) names are stored only as `first` and `last`, because the design can use photos rather than names to signal gender.
- All files are plain text, one name per line.

Paper source details:
- Black/White first names come from Table B1.
- Black/White last names come from Table B2.
